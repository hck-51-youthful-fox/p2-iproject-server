## PC-PEEKER

PC-PEEKER its forum for hardware PC enthusiast.

## Demo

https://drive.google.com/file/d/1TEp3qO8ytrFFLDkNRn4DyTUQ6SFy2k7D/view?usp=share_link

## Features

### Customer Side

- Login and Register
- Search thread
- Add new thread
- Pagination thread
- Add gif on comment thread
- Share on Facebook

## 3rd Party API

- NewsAPI
- Giphy

## Built With

- Vue Js
- Pinia State Management
- Firebase
- TailwindCSS

- NodeJS
- Express.js
- Postgres
- Sequelize
- Heroku
